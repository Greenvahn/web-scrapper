# web-scrapper
A basic web scrapper using node JS

## Technologies
* axios
* cheerio
* express
* nodemon



### Run from the terminal

1 - Target the URL's website from `index.js`
```
const URL = 'https://samplexyz.com'
```
2 - Target the website's components using `cheerio` from `index.js`

3 - Run the scrip
```
yarn start
```
